# SPL-Coding-Project
This repo contains all associated materials for the Structure of Programming Languages Coding Project
Programming languages used: C++ (object-oriented) and Python (procedural)
C++ compiler used: g++
Python Environment: 3.9.1

Different Concepts:
	Static typing (C++) vs dynamic typing (Python)
	Manual garbage collection (C++) vs generational garbage collection (Python)
	Call by value (C++) vs call by object reference (Python)
